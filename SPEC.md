# 第二大脑系统 (Second Brain) - 规格文档

## 1. 项目概述

- **项目名称**: Second Brain
- **类型**: Web 应用 (Next.js)
- **核心功能**: 统一管理笔记、对话记录和记忆碎片，支持全局搜索和多维筛选
- **目标用户**: 个人知识管理

## 2. UI/UX 规格

### 布局结构

- **顶部**: 简洁标题 + Cmd+K 搜索按钮
- **主体**: 左侧筛选栏 + 右侧内容列表
- **响应式**: 移动端隐藏侧边栏，改为顶部筛选

### 视觉设计

- **风格**: 极致简约 (Minimalist)
- **主色**: `#ffffff` (白) + `#f5f5f5` (浅灰背景)
- **文字**: `#333333` (主文字) + `#888888` (次要文字)
- **强调色**: `#0066ff` (蓝色，用于链接和按钮)
- **间距**: 16px 基础单位
- **字体**: 系统字体 (San Francisco / PingFang SC / Microsoft YaHei)

### 组件

1. **搜索栏 (Cmd+K)**
   - 居中弹出模态框
   - 支持搜索所有内容类型
   - 显示搜索结果预览

2. **筛选栏**
   - 日期选择器 (开始日期 - 结束日期)
   - 内容类型下拉 (全部 / 笔记 / 对话 / 记忆)

3. **记忆卡片**
   - 标题
   - 内容预览 (最多3行)
   - 标签 (类型、日期)
   - 点击展开详情

4. **空状态**
   - 无结果时显示友好提示

## 3. 功能规格

### 核心功能

1. **全量列表**
   - 分页加载 (每页20条)
   - 虚拟滚动优化性能

2. **全局搜索 (Cmd+K)**
   - 快捷键唤起搜索
   - 实时搜索 (debounce 300ms)
   - 搜索标题和内容

3. **多维筛选**
   - 日期范围筛选
   - 内容类型筛选
   - 筛选条件可组合

### 数据结构

```typescript
interface Memory {
  id: string;
  title: string;
  content: string;
  type: 'note' | 'conversation' | 'memory';
  tags: string[];
  createdAt: string;
  updatedAt: string;
}
```
